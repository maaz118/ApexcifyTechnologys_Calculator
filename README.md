# Calculator Web Application
## Project Overview
This project is a simple web-based calculator developed using HTML, CSS and JavaScript.  
It was created as part of my Web Development Internship at ApexcifyTechnologys.
The calculator allows users to perform basic arithmetic operations through an interactive and responsive user interface.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Addition, subtraction, multiplication and division operations
- Interactive calculator buttons
- Display screen for results
- Clear screen functionality
- Responsive design
- Hover button effects

## How to Run the Project
1. Download or clone this repository
2. Open the project folder
3. Run the file **index.html** in any web browser
